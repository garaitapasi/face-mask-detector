# 😷 Face Mask Detection System

## 📌 Description

This project is a **Face Mask Detection System** built using Deep Learning.
It can detect whether a person is wearing a mask or not from an image.

The model is trained using **MobileNetV2** and deployed using **Streamlit** for an easy-to-use web interface.

---

## 🚀 Features

* Upload an image and get instant prediction
* Detects **Mask 😷** or **No Mask ❌**
* Simple and user-friendly interface
* Built using deep learning techniques

---

## 🧠 Technologies Used

* Python
* TensorFlow / Keras
* OpenCV

* Streamlit
